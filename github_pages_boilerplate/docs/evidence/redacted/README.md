# Redacted Evidence Notes

This folder is reserved for sanitized evidence summaries that are safe for portfolio sharing.

Allowed content:
- Reproduction timelines without endpoint details
- High-level request sequence descriptions
- Impact and remediation rationale

Not allowed in this folder:
- Program or company identifiers
- Endpoint paths, parameters, object IDs, payloads
- Raw request or response dumps
- Private report IDs, links, or screenshots
